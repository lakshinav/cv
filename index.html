ML pipelines in production

During this course I've build a ML pipeline on the basis of my Jupyter notebook.
The notebook is devoted to the time series prediction, namely the prediction of unemployment in Russia, and was used in the student project at HSE University. The pipeline is built in Kedro environment and includes:

  • web-scraping step for collecting data (BeatifulSoap)
  • building prediction (pandas, statsmodels)
  • testing (hypothesis)
  • serving the result by API (fastAPI, uvicorn)
  • scheduling the pipeline tasks, e.g. web-scraping (airflow)
  • CI/CD (Docker, Github Actions)
  • evaluating metrics
